# MH4U Summer School Landing

Static landing page for MH4U Summer School 2026.

## Main File

The landing page lives here:

`project/MH4U Summer School - Landing.html`

The root `index.html` redirects to that page so GitHub Pages can open the site from the repository homepage.

## GitHub Pages

Recommended Pages settings:

- Source: deploy from branch
- Branch: `main`
- Folder: `/ (root)`

The repository intentionally ignores large design/reference uploads so the temporary public version only contains the landing page and required assets.
